## 三、数据模型

### TaskModel（表: `tasks`）

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | UUID str PK | 主键 |
| `channel_id` | FK → channels.id (SET NULL) | 可选所属频道 |
| `workspace_id` | str, nullable, indexed | 可选工作区 |
| `title` | str, NOT NULL, indexed | 任务标题 |
| `description` | text | 任务描述 |
| `status` | str, default=CREATED | 状态（7 个状态值） |
| `priority` | int, default=2 | 0~4 |
| `intent` | str | 预留意图分类 |
| `created_by` | str, NOT NULL | 创建者标识 |
| `created_at` / `updated_at` / `completed_at` | datetime | 时间戳 |

### TaskStepModel（表: `task_steps`）

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | UUID str PK | |
| `task_id` | FK → tasks.id (CASCADE) | 所属 Task |
| `teammate_id` | str, nullable | 执行 teammate |
| `order` | int, NOT NULL | 执行顺序 |
| `objective` / `input_context` / `output` | text | 目标/上下文/结果 |
| `status` | str, default=PENDING | 6 个状态值 |
| `maeos_task_id` | str, nullable | 关联 MAEOS task |
| `error` / `retry_count` | text / int | 错误信息/重试次数 |
| `started_at` / `completed_at` | datetime, nullable | 执行时间 |

### TaskExecutionModel（表: `task_executions`）

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | UUID str PK | |
| `task_step_id` | FK → task_steps.id (CASCADE) | 所属 Step |
| `maeos_task_id` / `trace_id` | str, nullable | 关联 MAEOS + Trace |
| `attempt` | int | 第几次尝试 |
| `execution_time_ms` / `token_usage` / `cost` / `output_snapshot` | int / text | 执行指标 |

### 级联关系

```
tasks ──CASCADE──► task_steps ──CASCADE──► task_executions
```

删除 Task → 自动删除所有 Step + Execution。

## 四、状态机

### Task 状态迁移

```text
CREATED ──► PLANNING ──► EXECUTING ──► COMPLETED
    │            │            │
    └──► CANCELLED    └──► FAILED ──► PLANNING (re-plan)
                              │
                         EXECUTING ◄── PAUSED
```

### TaskStep 状态迁移

```text
PENDING ──► SCHEDULED ──► RUNNING ──► COMPLETED
    │            │              │
    └──► SKIPPED          FAILED ──► PENDING (retry)
```

## 五、API 端点

### CRUD

| Method | Path | 说明 |
|---|---|---|
| `POST` | `/api/tasks` | 创建 Task |
| `GET` | `/api/tasks` | 列表（filter: channel_id, status, workspace_id） |
| `GET` | `/api/tasks/{id}` | 详情（含 Steps） |
| `PATCH` | `/api/tasks/{id}` | 更新元数据 |
| `DELETE` | `/api/tasks/{id}` | 删除 Task |

### Lifecycle

| Method | Path | 迁移 |
|---|---|---|
| `POST` | `/api/tasks/{id}/plan` | CREATED → PLANNING |
| `POST` | `/api/tasks/{id}/execute` | PLANNING → EXECUTING |
| `POST` | `/api/tasks/{id}/pause` | EXECUTING → PAUSED |
| `POST` | `/api/tasks/{id}/resume` | PAUSED → EXECUTING |
| `POST` | `/api/tasks/{id}/cancel` | any → CANCELLED |
| `POST` | `/api/tasks/{id}/complete` | EXECUTING → COMPLETED |
| `POST` | `/api/tasks/{id}/fail` | EXECUTING/PLANNING → FAILED |
